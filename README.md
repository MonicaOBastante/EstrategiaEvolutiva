# EstrategiaEvolutiva
Trabajo universitario realizado durante la optativa "Algoritmos Genéticos y Evolutivos"
Este programa esta diseño para ejecutarse numerosas veces con los mismos parametros y asi obtener estadisticas
y diferentes resultados, estando así hecho para una realizacion cómoda de pruebas.


Esta estrategia evolutiva coge la funcion fitness de una pagina web, teniendo control de errores
en caso de que la pagina diese problemas o se quedase colgada.

El problema al que va dirigido es a cuantos grados deben colocarse una serie de motores, que determinan el tamaño
del cromosoma.
